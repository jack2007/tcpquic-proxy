# Finished Docs

This directory stores completed design and implementation-plan documents.

The active `docs/` tree should keep documents that are still current references,
open plans, or ongoing investigation notes. Historical benchmark result
directories are left in place unless they are explicitly superseded.

Current grouping:

- `plans/`: completed legacy implementation plans.
- `specs/`: completed legacy design specs.
- `superpowers/plans/`: completed Superpowers execution plans.
- `superpowers/specs/`: completed Superpowers design specs.
